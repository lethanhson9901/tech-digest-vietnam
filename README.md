# TechDigest Vietnam

Trang web tổng hợp và dịch tự động tin tức công nghệ từ các blog quốc tế sang tiếng Việt.

## Tính năng chính

- Thu thập nội dung từ các blog công nghệ thông qua API hoặc RSS feed
- Tự động dịch nội dung từ tiếng Anh sang tiếng Việt
- Tạo bản tin tổng hợp hàng ngày
- Gửi email newsletter đến người đăng ký
- Lưu trữ nội dung trên trang web để dễ dàng truy cập
